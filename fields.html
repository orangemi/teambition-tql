# TQL Support Fields TQL所支持的字段

TQL支持的字段中除了text之外，其他都随着搜索类型不同而不同。下文会按照搜索类型给出该类型所支持的固定字段。

# Task

  • id: 任务ObjectId, 必须是ObjectId
  • text: 任务标题,备注,以及评论
  • title: 任务标题
  • titleSummary: 任务标题(按分词)
  • titleRaw: 任务标题(按字符)
  • content: 任务备注
  • activityContent: 任务评论
  • created: 任务创建时间
  • updated: 任务更新时间
  • startDate: 任务开始时间
  • dueDate: 任务截止时间
  • reminderDate: 任务提醒时间
  • accomplished: 任务完成时间
  • storyPoint: 任务storyPoint
  • progress: 任务完成度
  • sprintId: Sprint Id
  • scenarioId: scenarioId
  • tasklistId: tasklistId
  • stageId: stageId
  • rating: number
  • totalWorkTime: 任务计划工时
  • totalUsedTime: 任务提交工时
  • priority: 任务优先级(0,1,2)
  • isDone: 任务是否完成
  • isSubTask: 是否是自任务
  • isOnlyInvolveVisible: 是否参与者可见(boolean)
  • isArchived: 是否归档
  • creatorId: 任务创建者ObjectId
  • executorId: 任务执行者ObjectId
  • projectId: 任务所在Project的ObjectId
  • involveMembers: 任务参与者ObjectId
  • tagId: 任务标签ObjectId
